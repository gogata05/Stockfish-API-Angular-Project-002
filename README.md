# Stockfish-API

## Introduction

Stockfish-API is application for playing chess against Stockfish chess AI and friends.

## Features

- Stockfish API
- Github Actions Deploy

## How to use?

- 0.Download the repository and extract it to folder
- 1.Open with VS Code
- 2.npm install
- 3.npm start
- 4.Enjoy!
